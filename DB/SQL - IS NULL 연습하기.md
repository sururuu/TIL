# SQL - IS NULL 연습하기

### 이름이 없는 동물의 아이디

```SQL
SELECT ANIMAL_ID FROM ANIMAL_INS
WHERE NAME IS NULL
ORDER BY ANIMAL_ID
```



### 이름이 있는 동물의 아이디

```SQL
SELECT ANIMAL_ID FROM ANIMAL_INS
WHERE NAME IS NOT NULL
ORDER BY ANIMAL_ID
```



### NULL 처리하기

```SQL
SELECT ANIMAL_TYPE, IFNULL(NAME,'NO name') AS NAME, SEX_UPON_INTAKE FROM ANIMAL_INS
```
